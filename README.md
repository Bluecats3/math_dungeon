# Math Dungeon Escape 

A simple Python game where players solve math problems to escape a dungeon.  
Each correct answer moves you forward, while wrong answers reduce your health.

---

## How the Game Works

- You start with **3 health points**
- You must pass through **5 rooms**
- Each room gives you a random math problem:
  - Addition
  - Subtraction
  - Multiplication
- If you answer correctly → you move forward  
- If you answer incorrectly → you lose health
  
 Reach the final room to escape the dungeon!

---

## How to Run

Make sure Python is installed, then run:
    python math_dungeon_escape.py

## Features

- Random math questions every playthrough  
- Health system (3 lives)  
- Multiple difficulty levels through different operations  
- Simple game loop with win/lose conditions
